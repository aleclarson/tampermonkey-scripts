# Tampermonkey Userscripts

A small collection of self-contained Tampermonkey userscripts.

## Scripts

- `npmjs-to-npmx.js` — Redirects npm package overview pages from npmjs.com/npmjs.org to the matching npmx.dev page, while avoiding redirect loops and non-readme tabs.
- `npmx-preserve-npm-referrer.js` — Removes `noreferrer` from npm package links on npmx.dev so npmjs pages can detect npmx.dev as the referrer.
- `x-highlight.js` — Highlights X/Twitter posts with a low comment-to-view ratio using a green border.
- `x-no-lames.js` — Fades or hides low-like X/Twitter replies on status, search, and list pages.
- `x-shift-block.js` — Adds Shift+Click blocking for the X/Twitter post under the cursor.
- `youtube-enhanced-interaction.js` — Adds Alt+Click shortcuts for YouTube recommendations to mark videos as “Not interested” or “Don’t recommend channel”.
- `youtube-hide-short-replies.js` — Hides short YouTube comments with too few replies and provides a reveal counter/button.
